        # Detailed Description

        This module provides additional helper methods such as JavaScript like var (ffw::Any), JavaScript like Promise (ffw::Promise), thread safe no allocation ring buffer, string functions, text wrapper, and a string tokenizer wiuth utf-8 support. This library uses utfcpp
